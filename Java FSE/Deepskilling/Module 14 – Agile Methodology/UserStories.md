# Agile User Stories

## User Story 1 – Student Login

**As a** Student

**I want** to log in to the portal

**So that** I can access my dashboard.

### Acceptance Criteria

Given the student has a valid account

When the student enters valid credentials

Then the dashboard should be displayed.

---

## User Story 2 – Course Enrollment

**As a** Student

**I want** to enroll in a course

**So that** I can start learning.

### Acceptance Criteria

Given the course is available

When the student clicks Enroll

Then the course should be added.

---

## User Story 3 – Assignment Submission

**As a** Student

**I want** to upload my assignment

**So that** the instructor can evaluate it.

### Acceptance Criteria

Given the assignment deadline is active

When the student uploads a valid file

Then the submission should be successful.

---
