## How to Run the Project

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to the Project

```bash
cd react-debug-demo
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```

(or another available port if 3000 is already in use.)
