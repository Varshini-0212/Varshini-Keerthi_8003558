# Module 9 – Single Page Application Framework (React)

## Overview

This repository contains the hands-on exercises completed as part of **Module 9 – Single Page Application Framework (React)** in the **Cognizant Digital Nurture 4.0 – Java Full Stack Engineer (FSE)** program.

The module introduces the fundamentals of React, including component-based architecture, JSX, state management, props, event handling, conditional rendering, forms, REST API integration, Context API, unit testing, and mocking.

## How to Run a Project

Navigate to the required exercise folder.

```bash
cd ExerciseXX_ProjectName
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open your browser:

```
http://localhost:3000
```

(If port 3000 is already in use, React will automatically suggest another available port.)

---

## Running Unit Tests

To execute Jest tests:

```bash
npm test
```

Press:

```
a
```

to run all test cases.
