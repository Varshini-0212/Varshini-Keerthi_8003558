# AWS Services

## Amazon EC2

Virtual machine service for hosting applications.

---

## Amazon S3

Object storage service for storing files and backups.

---

## Amazon EBS

Persistent block storage for EC2 instances.

---

## Amazon VPC

Creates an isolated virtual network in AWS.

---

## IAM

Identity and Access Management for users, roles, and permissions.

---

## Amazon RDS

Managed relational database service.

Supports:
- MySQL
- PostgreSQL
- MariaDB
- Oracle
- SQL Server

---

## DynamoDB

Fully managed NoSQL database service.

---

## AWS Lambda

Serverless computing service that runs code without managing servers.

---

## API Gateway

Creates and manages REST APIs.

---

## CloudFormation

Infrastructure as Code service for provisioning AWS resources.

---

## CloudWatch

Monitoring and logging service.

---

## AWS Pricing Models

- Free Tier
- Pay As You Go
- Reserved Instances
- Spot Instances
