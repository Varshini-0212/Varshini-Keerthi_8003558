# Prompt Engineering

## What is Prompt Engineering?

Prompt Engineering is the process of designing effective prompts to obtain accurate and useful responses from Generative AI models.

## Types of Prompting

### Zero-shot Prompting

The model answers without any examples.

Example:

Prompt:
Summarize the importance of Agile methodology.

---

### One-shot Prompting

The model receives one example before answering.

Example:

Example:
Input: Java
Output: Programming Language

Input: Python

---

### Few-shot Prompting

The model receives multiple examples before answering.

Example:

Input: Dog
Output: Animal

Input: Rose
Output: Flower

Input: Mango

---

## Best Practices

- Write clear prompts.
- Provide context.
- Mention the expected output format.
- Use examples when necessary.
- Break complex tasks into smaller prompts.

---

## AI Hallucination

AI Hallucination occurs when an AI model generates incorrect or fabricated information confidently.
